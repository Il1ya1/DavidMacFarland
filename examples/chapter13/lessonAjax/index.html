Ajax

Введение

JavaScript - это мощное но не всесильное средство. Если вы ходите предсавить информацию из базы данных, отправить форму с заполнеными данными, загрузить новый html на страницу, поставить оценку в рейтинге и тд. вам потребуется обратиться к серверу, Ajax позволяет сделать это без перезагрузки сервера.

Что такое Ajax

Ajax - это аббревиатура от Asynchronous JavaScript и XML (Асинхронный JavaScript и XML)

Ajax не является отдельной технологией как JavaScript или CSS это набор технологий - языка JavaScript, браузера и веб себсера для получения и отображения обновленной информации без загрузки новой страницы.

Коротко говоря современные браузеры позволябт благодоря коду JavaScript посылать запрос на сервер, который присылает ответ браузеру некоторые данные JavaScript берез эти сведения и что нибудь с ними делает.
